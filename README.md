# Lara12 Complete

This is a starter kit for [Laravel](https://laravel.com) web applications. It uses the [Livewire](https://laravel-livewire.com) framework and [Tailwind CSS](https://tailwindcss.com) for styling.

## Packages included

- [Livewire](https://livewire.laravel.com)
- [Volt](https://livewire.laravel.com/docs/volt)
- [Laravel Pulse](https://laravel.com/docs/12.x/pulse)
- [Laravel Telescope](https://laravel.com/docs/12.x/telescope)
- [Laravel Debugbar](https://github.com/barryvdh/laravel-debugbar)
- [Laradumps](https://laradumps.dev/)
- [archtechx/enums] - Trait helpers for enums
- prettier and prettier-plugin-blade@^2 for formatting blade files with Voltage Extension
